## 如何学习前端?

!!! tip

    可能大多数好的学习方法就是这样(难度递增)：

    - 看优质视频
    - 读官方文档
    - 学习开源项目
    - 做开源项目
    - ... 


你也需要选择适合自己的学习方法



!!! abstract "我的建议"

    长时间学习一些东西是非常无聊的, 你可以尝试==速通三件套==, 再学react, 在这个过程中, 我建议穿插学习算法, 刷题
    如果你没有学习数据结构, 可以先学习一下数据结构(推荐名课 cs61b)
