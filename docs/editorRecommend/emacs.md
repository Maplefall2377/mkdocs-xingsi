# Emacs

- **推荐指数**: 🌟🌟🌟
- **优点**: 如果你能熟练使用，你完全可以节省所有使用鼠标的时间，你可以将一切放到Emacs中完成
- **缺点**: 你需要定义（不是记忆）非常多的快捷键，花费很多很多时间customize

## Emacs是什么

Emacs 不只是一个编辑器，因为它能做的实在太多了，有人说Emacs表面上是一个编辑器，实际上是一个操作系统。在Emacs里面，你可以写代码，制订计划，听歌，看图，发邮件，写博客，上网冲浪等等。Emacs不是一个开箱即用的软件，很多功能需要自己写。如果你只是需要一个写代码的工具，Emacs可能不是最好的选择。并不是Emacs不能写，而是需要付出很多的时间和精力来完善Emacs。Emacs可以是一种生活方式，它能让你的生活更加轻松。

## 配置

Emacs 的配置文件在主目录的`.emacs`文件夹下，Emacs 启动时会运行配置文件夹下的`init.el`文件。
el文件是用`emacs lisp`语言写的，这个语言是lisp语言的一种方言，语法非常奇特，很难学。

新手建议先直接复制大佬的配置或者用现成的`spacemacs`和`doom emacs`，熟悉emacs的哲学之后再配置自己的。Emacs 用户的一大乐趣就是不断更新自己的配置，这是很有成就感的事情。
